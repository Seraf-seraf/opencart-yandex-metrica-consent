# Yandex Metrica Consent for OpenCart 3

Самостоятельное расширение для OpenCart 3, которое подключает Яндекс Метрику только после явного согласия посетителя.

## Возможности

- ID счетчика, Вебвизор, ecommerce и статус в административной панели.
- Настраиваемые по языкам заголовок, описание и подписи элементов cookie-плашки.
- Выбор CMS-страницы политики конфиденциальности.
- Настраиваемый срок хранения согласия от 1 до 730 дней.
- Отказ не блокирует работу магазина и удаляет доступные cookies `_ym_*`.
- Повторное открытие настроек аналитики из footer.
- Цели и ecommerce активируются только после согласия.

## Установка

1. Соберите ZIP так, чтобы каталог `upload` находился в корне архива.
2. Загрузите архив через `Расширения → Установка расширений`.
3. Откройте `Расширения → Расширения`, выберите тип `Аналитика` и установите `Яндекс Метрика и cookie-согласие`.
4. Откройте форму редактирования расширения, заполните настройки и включите статус.

Расширение не требует изменений темы: плашка и кнопка повторной настройки создаются собственным JavaScript.
